## MH-Crutches 
- you can put this in your ambulance vehicle items. do dochters can give players a crutch. 


## Inventory Image
- place `crutch.png` into your `[qn]/qb-inventory/image` folder


## Shared Item (EN)
```lua
["crutch"] = {
    ["name"] = "crutch",
    ["label"] = "Een Kruk".
    ["weight"] = 1000,
    ["type"] = "item",
    ["image"] = "crutch.png",
    ["unique"] = false,
    ["useable"] = true,
    ["shouldClose"] = true,
    ["combinable"] = nil,
    ["description"] = "A crutch, if you have a hard time walking!"
},
```
